# Ludo RPG

An experimental card based RPG-style expansion of classic Ludo, built with Flutter.

## Status
Alpha (v0.1.0)

## Features
- Card-based mechanics (Attack / Defence / Movement / Board)
- Animated fan hand
- Draw & discard piles
- Desktop-first prototype

## Roadmap
- Fix hover animation glitches
- Add card effects
- Board integration
